# simmons-testbed-android

[동국대] 테스트베드 데이터 수집/활용/확산을 위한 스마트캠퍼스 해커톤

## 프로젝트 소개
**주제 : IoT 센서를 통한 청각장애인 부모를 위한 영아 케어 서비스 및 애플리케이션**
